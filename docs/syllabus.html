Ementa

Neste curso você irá aprender os fundamentos de uma das principais linguagens para ciência de dados. Além disso você terá um curso completo de PostgreSQL: O banco de dados relacional Open Source mais avançado do mundo. Melhor ainda, você aprenderá a manusear bases de dados relacionais apartir do R.
